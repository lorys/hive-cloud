What's Hive Cloud ?

Hive cloud is an open source, decentralized cloud storage.

How does Hive Cloud work ?

Hive cloud lives in each browser connected to a hive cloud server. Each browser stores a specific amount of chunks.

You can configure how many chunks you allow the Hive to use. 1 chunk = bytes.
